# BDNR
Comparación de metodologías de diseño de bases de datos de documentos

Para la ejecución de los Scripts para la carga de los datos se deberá descargar de https://www.kaggle.com/rounakbanik/the-movies-dataset?select=movies_metadata.csv el conjunto de datos. Luego se deberán tranformar los archivos credits.csv, movies_metadata.csv y ratings_small.csv a JSON, para esto se puede utilizar la herramienta MongoDB Compass.
Finalmente los archivos JSON resultantes se deben pegar en las carpetas donde se encuentras los scripts (Scripts de Carga de Datos - Hoberman o Scripts de Carga de Datos - Mongo).
